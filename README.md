# Negocio Julio — Web estática (Vite + React + Tailwind)

## Requisitos
- Node.js 18+ (recomendado 20+)
- VSCode

## Arranque rápido
```bash
npm install
npm run dev
```

## Build para producción
```bash
npm run build
npm run preview
```

## Qué incluye
- Secciones: Inicio, Sobre, Servicios, Ubicación (Google Maps), Contacto
- Botones directos: WhatsApp + Email
- Diseño responsive con Tailwind
